## 1.2 核心設計原則

### 1.2.1 分層解耦原則
```
┌────────────────────────────────────────┐
│  每一層只依賴下層，不直接跳層調用       │
│  層間通過定義良好的接口通信             │
│  允許獨立擴展和替換各層實現            │
└────────────────────────────────────────┘
```

### 1.2.2 事件驅動架構
```
┌────────────────────────────────────────┐
│  所有Agent間通信通過事件總線            │
│  支持異步處理和鬆耦合協作               │
│  便於追蹤和審計所有交互                │
└────────────────────────────────────────┘
```

### 1.2.3 狀態機驅動
```
┌────────────────────────────────────────┐
│  使用LangGraph定義所有工作流狀態機      │
│  明確的狀態轉換和條件判斷               │
│  支持中斷恢復和長期運行                │
└────────────────────────────────────────┘
```

# 第二部分：Agent協作拓撲詳細設計

## 2.1 Agent分類與職責矩陣

### 2.1.1 領域專家Agent (Domain Expert Agents)

| Agent名稱 | 核心職責 | 輸入 | 輸出 | 觸發條件 |
|-----------|----------|------|------|----------|
| **CareerAgent** | 職涯規劃、技能評估、轉職建議 | 用戶背景、職業目標、市場數據 | 職涯路徑圖、學習計劃、行動清單 | 用戶提及工作、職涯相關話題 |
| **FinanceAgent** | 財務規劃、預算管理、投資建議 | 收入支出數據、財務目標、風險偏好 | 預算方案、儲蓄計劃、投資組合 | 用戶提及金錢、財務決策 |
| **SkillAgent** | 技能學習路徑、學習方法優化 | 當前技能、目標技能、時間資源 | 學習計劃、資源推薦、進度追蹤 | 用戶想學習新技能 |
| **HealthAgent** | 身心健康、作息管理、壓力調適 | 健康數據、生活習慣、症狀描述 | 健康計劃、作息建議、預警信號 | 用戶提及健康問題或目標 |
| **RelationshipAgent** | 人際關係、溝通技巧、社交策略 | 關係情境、溝通記錄、情感狀態 | 溝通建議、關係修復策略 | 用戶提及人際困擾 |
| **DecisionAgent** | 重大決策分析、選項評估 | 決策情境、可選方案、評估標準 | 決策分析報告、推薦選項 | 用戶面臨重大選擇 |

### 2.1.2 共享服務Agent (Shared Service Agents)

| Agent名稱 | 核心職責 | 輸入 | 輸出 | 觸發條件 |
|-----------|----------|------|------|----------|
| **DataCollector** | 搜尋權威資料、驗證信息來源 | 查詢主題、領域標籤 | 結構化知識、來源引用 | 需要外部知識支持時 |
| **ContextMemory** | 管理用戶畫像、檢索相關記憶 | 用戶ID、查詢意圖、記憶類型 | 相關記憶片段、用戶畫像 | 每次交互都觸發 |
| **ProgressTracker** | 追蹤KBI指標、生成進度報告 | 行動記錄、目標設定、時間範圍 | 進度儀表板、達成分析 | 定期執行或用戶查詢 |
| **ConflictResolver** | 解決Agent間建議衝突 | 多個Agent建議、衝突點分析 | 統一建議、衝突解釋 | 檢測到建議衝突時 |

### 2.1.3 監督Agent (Supervisory Agents)

| Agent名稱 | 核心職責 | 輸入 | 輸出 | 觸發條件 |
|-----------|----------|------|------|----------|
| **KBIMonitor** | 監控關鍵行為指標、檢測異常 | 行為數據、KBI閾值 | 異常警報、干預建議 | 實時監控或定期檢查 |
| **Intervention** | 執行主動干預、發送提醒 | 觸發條件、用戶狀態、干預策略 | 干預消息、跟進計劃 | 檢測到需要干預時 |
| **EthicsGuardian** | 確保建議符合倫理、檢測風險 | Agent建議、用戶狀態、風險規則 | 風險評估、修改建議 | 所有建議輸出前 |
| **QualityAssurance** | 評估回應質量、檢測幻覺 | 最終輸出、知識來源 | 質量評分、改進建議 | 輸出發送前 |

### 2.4.3 衝突解決Agent實現

```python
class ConflictResolverAgent:
    """
    專門負責解決Agent間建議衝突
    """
    
    async def resolve_conflicts(
        self,
        recommendations: List[AgentRecommendation],
        user_context: UserContext,
        conflict_type: ConflictType
    ) -> ResolutionResult:
        
        # 步驟1: 分析衝突本質
        conflict_analysis = await self._analyze_conflict(
            recommendations, conflict_type
        )
        
        # 步驟2: 收集額外上下文
        enriched_context = await self._enrich_context(
            user_context, conflict_analysis
        )
        
        # 步驟3: 應用解決策略
        resolution_strategy = self._select_strategy(conflict_type)
        
        # 步驟4: 生成解決方案
        resolution = await self._generate_resolution(
            recommendations,
            conflict_analysis,
            enriched_context,
            resolution_strategy
        )
        
        # 步驟5: 驗證解決方案
        validation = await self._validate_resolution(
            resolution, user_context
        )
        
        return ResolutionResult(
            resolution=resolution,
            explanation=validation.explanation,
            confidence=validation.confidence,
            alternatives=validation.alternatives if validation.confidence < 0.8 else None
        )
```

### 3.2.2 記憶檢索服務實現

```python
class MemoryRetriever:
    """
    統一的記憶檢索服務
    """
    
    def __init__(self):
        self.stm_store = RedisMemoryStore()      # 短期記憶
        self.mtm_store = PostgreSQLMemoryStore() # 中期記憶
        self.ltm_store = ObjectStore()           # 長期記憶
        self.vector_store = QdrantStore()        # 向量記憶
    
    async def retrieve_memories(
        self,
        query: str,
        user_id: str,
        context: RetrievalContext,
        config: RetrievalConfig = None
    ) -> RetrievedMemories:
        """
        多層次記憶檢索主入口
        """
        config = config or RetrievalConfig.default()
        
        # 並行檢索所有記憶層
        results = await asyncio.gather(
            self._retrieve_stm(user_id, context),
            self._retrieve_mtm(user_id, query, context),
            self._retrieve_ltm(user_id, query),
            self._retrieve_vector(user_id, query, config.top_k)
        )
        
        stm_memories, mtm_memories, ltm_memories, vector_memories = results
        
        # 融合所有記憶
        fused_memories = self._fuse_memories(
            stm_memories, mtm_memories, ltm_memories, vector_memories
        )
        
        # 排序和過濾
        ranked_memories = self._rank_memories(fused_memories, query, config)
        
        # 生成摘要
        summary = await self._generate_summary(ranked_memories, query)
        
        return RetrievedMemories(
            memories=ranked_memories[:config.max_results],
            summary=summary,
            metadata={
                "sources": {
                    "stm": len(stm_memories),
                    "mtm": len(mtm_memories),
                    "ltm": len(ltm_memories),
                    "vector": len(vector_memories)
                },
                "retrieval_time": time.time()
            }
        )
    
    async def _retrieve_stm(
        self, 
        user_id: str, 
        context: RetrievalContext
    ) -> List[Memory]:
        """檢索短期記憶"""
        session_key = f"session:{user_id}:{context.session_id}"
        
        # 獲取對話上下文
        messages = await self.stm_store.get_messages(session_key, limit=20)
        
        # 獲取會話狀態
        session_state = await self.stm_store.get_state(session_key)
        
        return self._parse_stm_memories(messages, session_state)
    
    async def _retrieve_mtm(
        self, 
        user_id: str, 
        query: str, 
        context: RetrievalContext
    ) -> List[Memory]:
        """檢索中期記憶"""
        memories = []
        
        # 檢索行為模式
        patterns = await self.mtm_store.query_patterns(
            user_id=user_id,
            pattern_types=context.relevant_domains,
            time_range=context.time_range
        )
        memories.extend(patterns)
        
        # 檢索用戶偏好
        preferences = await self.mtm_store.get_preferences(user_id)
        memories.extend(preferences)
        
        # 檢索週期性規律
        cycles = await self.mtm_store.get_cyclical_patterns(user_id)
        memories.extend(cycles)
        
        return memories
    
    async def _retrieve_vector(
        self, 
        user_id: str, 
        query: str, 
        top_k: int
    ) -> List[Memory]:
        """向量語義檢索"""
        # 生成查詢嵌入
        query_embedding = await self.embedding_service.embed(query)
        
        # 執行向量搜索
        results = await self.vector_store.search(
            collection="conversation_embeddings",
            vector=query_embedding,
            filter={"user_id": user_id},
            top_k=top_k,
            score_threshold=0.7
        )
        
        return self._parse_vector_memories(results)
```

```python
class MemoryDecayManager:
    """
    記憶衰減與清理管理器
    """
    
    # 衰減配置
    DECAY_CONFIG = {
        "short_term": {
            "ttl_hours": 24,
            "decay_function": "hard_cutoff",  # 硬截斷
            "cleanup_schedule": "continuous"
        },
        "medium_term": {
            "ttl_days": 90,
            "decay_function": "exponential",  # 指數衰減
            "decay_rate": 0.01,  # 每日衰減1%
            "compression_schedule": "weekly",
            "cleanup_schedule": "daily"
        },
        "long_term": {
            "ttl": "permanent",
            "decay_function": "none",
            "consolidation_schedule": "monthly",
            "archive_threshold_years": 5
        }
    }
    
    async def apply_decay(self, memory_layer: str):
        """應用記憶衰減"""
        config = self.DECAY_CONFIG[memory_layer]
        
        if memory_layer == "medium_term":
            await self._apply_exponential_decay(config)
        
        await self._cleanup_expired_memories(memory_layer, config)
    
    async def _apply_exponential_decay(self, config: Dict):
        """指數衰減中期記憶"""
        # 獲取所有中期記憶
        memories = await self.mtm_store.get_all_memories()
        
        for memory in memories:
            # 計算當前重要性
            days_since_update = (now - memory.last_accessed).days
            decay_factor = (1 - config["decay_rate"]) ** days_since_update
            
            new_importance = memory.importance_score * decay_factor
            
            if new_importance < 0.2:
                # 重要性太低，考慮刪除或歸檔
                await self._archive_or_delete(memory)
            else:
                # 更新重要性分數
                await self.mtm_store.update_importance(
                    memory.id, new_importance
                )
    
    async def consolidate_memories(self):
        """
        記憶鞏固: 將多個相關記憶合併為更高層次的抽象
        """
        # 識別可合併的記憶群組
        groups = await self._identify_consolidation_groups()
        
        for group in groups:
            # 生成抽象記憶
            abstract_memory = await self._generate_abstraction(group)
            
            # 存儲抽象記憶
            await self.ltm_store.store_abstraction(abstract_memory)
            
            # 標記原始記憶為已整合
            for memory in group:
                await self.mtm_store.mark_consolidated(memory.id)
```

## 4.2 多Agent輸出整合機制

### 4.2.1 整合策略

```python
class DecisionFusionEngine:
    """
    決策融合引擎: 整合多個Agent的輸出
    """
    
    def __init__(self):
        self.aggregation_strategies = {
            "weighted_average": WeightedAverageStrategy(),
            "confidence_based": ConfidenceBasedStrategy(),
            "hierarchical": HierarchicalStrategy(),
            "consensus": ConsensusStrategy()
        }
    
    async def fuse_decisions(
        self,
        agent_outputs: List[AgentOutput],
        user_context: UserContext,
        fusion_config: FusionConfig
    ) -> FusedDecision:
        """
        融合多個Agent的決策輸出
        """
        
        # 步驟1: 標準化輸出格式
        standardized = self._standardize_outputs(agent_outputs)
        
        # 步驟2: 評估每個輸出的質量
        quality_scores = await self._assess_quality(standardized)
        
        # 步驟3: 檢測衝突
        conflicts = self._detect_conflicts(standardized)
        
        # 步驟4: 選擇並應用融合策略
        strategy = self._select_strategy(fusion_config, conflicts)
        fused = await strategy.fuse(standardized, quality_scores, user_context)
        
        # 步驟5: 個性化調整
        personalized = self._personalize(fused, user_context)
        
        # 步驟6: 生成置信度評估
        confidence = self._calculate_confidence(
            standardized, quality_scores, conflicts
        )
        
        return FusedDecision(
            recommendation=personalized,
            confidence=confidence,
            contributing_agents=[o.agent_id for o in agent_outputs],
            conflict_summary=conflicts if conflicts else None,
            alternative_options=self._generate_alternatives(
                standardized, personalized
            )
        )
    
    def _select_strategy(
        self, 
        config: FusionConfig, 
        conflicts: List[Conflict]
    ) -> FusionStrategy:
        """根據情況選擇融合策略"""
        
        if conflicts and len(conflicts) > 2:
            # 衝突較多，使用共識策略
            return self.aggregation_strategies["consensus"]
        
        elif config.prefer_confidence:
            # 優先置信度
            return self.aggregation_strategies["confidence_based"]
        
        elif config.domain_hierarchy:
            # 有明確領域優先級
            return self.aggregation_strategies["hierarchical"]
        
        else:
            # 默認加權平均
            return self.aggregation_strategies["weighted_average"]


class ConfidenceBasedStrategy(FusionStrategy):
    """
    基於置信度的融合策略
    """
    
    async def fuse(
        self,
        outputs: List[StandardizedOutput],
        quality_scores: Dict[str, float],
        user_context: UserContext
    ) -> FusedRecommendation:
        
        # 計算加權分數 (置信度 * 質量分數)
        weighted_scores = []
        for output in outputs:
            weight = output.confidence * quality_scores.get(output.agent_id, 0.5)
            weighted_scores.append((output, weight))
        
        # 按權重排序
        weighted_scores.sort(key=lambda x: x[1], reverse=True)
        
        # 選擇最高權重的作為主要建議
        primary = weighted_scores[0][0]
        
        # 整合其他建議的補充觀點
        supplementary = []
        for output, weight in weighted_scores[1:]:
            if weight > 0.3:  # 只考慮權重足夠的
                supplementary.append({
                    "agent": output.agent_id,
                    "suggestion": output.suggestion,
                    "weight": weight
                })
        
        return FusedRecommendation(
            primary_recommendation=primary.suggestion,
            supplementary_points=supplementary,
            fusion_method="confidence_based"
        )
```

### 4.2.2 置信度計算模型

```python
class ConfidenceCalculator:
    """
    決策置信度計算器
    """
    
    def calculate(
        self,
        agent_outputs: List[AgentOutput],
        quality_scores: Dict[str, float],
        conflicts: List[Conflict],
        user_context: UserContext
    ) -> ConfidenceScore:
        """
        計算最終決策的置信度
        """
        
        # 因子1: Agent置信度一致性
        agent_confidences = [o.confidence for o in agent_outputs]
        consistency_score = self._calculate_consistency(agent_confidences)
        
        # 因子2: 建議質量
        avg_quality = sum(quality_scores.values()) / len(quality_scores)
        
        # 因子3: 衝突程度 (衝突越多，置信度越低)
        conflict_penalty = self._calculate_conflict_penalty(conflicts)
        
        # 因子4: 記憶支持度
        memory_support = self._assess_memory_support(agent_outputs, user_context)
        
        # 因子5: 知識來源可靠性
        source_reliability = self._assess_source_reliability(agent_outputs)
        
        # 加權組合
        weights = {
            "consistency": 0.25,
            "quality": 0.25,
            "conflict": 0.20,
            "memory": 0.15,
            "source": 0.15
        }
        
        final_confidence = (
            weights["consistency"] * consistency_score +
            weights["quality"] * avg_quality +
            weights["conflict"] * (1 - conflict_penalty) +
            weights["memory"] * memory_support +
            weights["source"] * source_reliability
        )
        
        return ConfidenceScore(
            overall=round(final_confidence, 2),
            breakdown={
                "consistency": round(consistency_score, 2),
                "quality": round(avg_quality, 2),
                "conflict_penalty": round(conflict_penalty, 2),
                "memory_support": round(memory_support, 2),
                "source_reliability": round(source_reliability, 2)
            },
            interpretation=self._interpret_confidence(final_confidence)
        )
    
    def _interpret_confidence(self, score: float) -> str:
        """解釋置信度分數"""
        if score >= 0.9:
            return "高度確信 - 建議可靠，可放心執行"
        elif score >= 0.7:
            return "較為確信 - 建議合理，但建議進一步確認"
        elif score >= 0.5:
            return "中等確信 - 建議有參考價值，需要更多討論"
        else:
            return "確信度低 - 信息不足，建議補充後再決策"
```

```python
class ExplainabilityGenerator:
    """
    決策可解釋性生成器
    """
    
    async def generate_explanation(
        self,
        fused_decision: FusedDecision,
        agent_outputs: List[AgentOutput],
        user_context: UserContext,
        memory_refs: List[MemoryReference]
    ) -> Explanation:
        """
        生成決策的完整解釋
        """
        
        explanation_parts = []
        
        # 1. 生成決策理由
        reasoning = await self._generate_reasoning(
            fused_decision, agent_outputs
        )
        explanation_parts.append(("reasoning", reasoning))
        
        # 2. 生成依據來源
        sources = self._generate_sources(
            agent_outputs, memory_refs
        )
        explanation_parts.append(("sources", sources))
        
        # 3. 生成預期效果
        effects = await self._generate_expected_effects(
            fused_decision, user_context
        )
        explanation_parts.append(("expected_effects", effects))
        
        # 4. 生成風險提示
        risks = self._generate_risk_warnings(
            fused_decision, agent_outputs
        )
        explanation_parts.append(("risks", risks))
        
        # 5. 生成替代選項
        alternatives = self._generate_alternatives(
            fused_decision, agent_outputs
        )
        explanation_parts.append(("alternatives", alternatives))
        
        return Explanation(
            parts=dict(explanation_parts),
            confidence=fused_decision.confidence,
            transparency_score=self._calculate_transparency(
                explanation_parts
            )
        )
    
    def _generate_sources(
        self,
        agent_outputs: List[AgentOutput],
        memory_refs: List[MemoryReference]
    ) -> List[SourceAttribution]:
        """生成來源歸因"""
        
        sources = []
        
        # Agent來源
        for output in agent_outputs:
            if output.contribution_weight > 0.2:
                sources.append(SourceAttribution(
                    type="agent",
                    source_id=output.agent_id,
                    description=f"{output.agent_id}的專業分析",
                    contribution_weight=output.contribution_weight,
                    confidence=output.confidence
                ))
        
        # 記憶來源
        for ref in memory_refs:
            sources.append(SourceAttribution(
                type="memory",
                source_id=ref.memory_id,
                description=ref.summary,
                relevance_score=ref.relevance,
                timestamp=ref.timestamp
            ))
        
        # 知識來源
        for output in agent_outputs:
            for knowledge in output.knowledge_sources:
                sources.append(SourceAttribution(
                    type="knowledge",
                    source_id=knowledge.source_id,
                    description=knowledge.title,
                    reliability=knowledge.reliability_score,
                    url=knowledge.url
                ))
        
        return sorted(sources, key=lambda s: s.contribution_weight, reverse=True)
```

### 5.2.1 核心狀態圖定義

```python
from langgraph.graph import StateGraph, END
from typing import TypedDict, Annotated, List
import operator

# 定義系統狀態
class SystemState(TypedDict):
    # 用戶輸入
    user_input: str
    user_id: str
    session_id: str
    
    # 處理狀態
    current_phase: str
    detected_intent: dict
    involved_domains: List[str]
    
    # Agent輸出
    agent_outputs: Annotated[List[dict], operator.add]
    
    # 記憶相關
    retrieved_memories: dict
    memory_context: str
    
    # 決策相關
    fused_decision: dict
    final_response: str
    
    # 元數據
    execution_trace: List[str]
    error_info: dict

# 創建狀態圖
workflow = StateGraph(SystemState)

# 定義節點
workflow.add_node("intent_classifier", intent_classification_node)
workflow.add_node("memory_retriever", memory_retrieval_node)
workflow.add_node("domain_router", domain_routing_node)
workflow.add_node("single_domain_processor", single_domain_node)
workflow.add_node("multi_domain_coordinator", multi_domain_node)
workflow.add_node("conflict_resolver", conflict_resolution_node)
workflow.add_node("decision_fusion", decision_fusion_node)
workflow.add_node("ethics_guardian", ethics_check_node)
workflow.add_node("quality_assurance", quality_check_node)
workflow.add_node("response_generator", response_generation_node)

# 定義邊和條件
def route_by_complexity(state: SystemState):
    """根據問題複雜度路由"""
    if state["detected_intent"]["urgency"] == "critical":
        return "emergency_response"
    elif len(state["involved_domains"]) == 1:
        return "single_domain_processor"
    else:
        return "multi_domain_coordinator"

workflow.add_conditional_edges(
    "domain_router",
    route_by_complexity,
    {
        "single_domain_processor": "single_domain_processor",
        "multi_domain_coordinator": "multi_domain_coordinator"
    }
)

# 設置入口和結束
workflow.set_entry_point("intent_classifier")
workflow.add_edge("response_generator", END)

# 編譯圖
app = workflow.compile()

# 執行
result = app.invoke({
    "user_input": "我想轉職但擔心財務問題",
    "user_id": "user_123",
    "session_id": "session_456"
})
```

```python
async def multi_domain_node(state: SystemState) -> SystemState:
    """
    多領域協作處理節點
    """
    involved_domains = state["involved_domains"]
    
    # 並行執行所有相關Domain Agent
    agent_tasks = []
    for domain in involved_domains:
        agent = get_domain_agent(domain)
        task = agent.process(
            input=state["user_input"],
            context=state["memory_context"],
            state=state
        )
        agent_tasks.append(task)
    
    # 等待所有Agent完成
    agent_results = await asyncio.gather(*agent_tasks, return_exceptions=True)
    
    # 處理結果
    outputs = []
    for domain, result in zip(involved_domains, agent_results):
        if isinstance(result, Exception):
            # 記錄錯誤但繼續處理
            state["execution_trace"].append(f"{domain}_agent_failed: {str(result)}")
            continue
        outputs.append({
            "agent_id": domain,
            "output": result,
            "timestamp": datetime.now().isoformat()
        })
    
    return {
        "agent_outputs": outputs,
        "execution_trace": state["execution_trace"] + ["multi_domain_complete"]
    }

async def conflict_resolution_node(state: SystemState) -> SystemState:
    """
    衝突解決節點
    """
    agent_outputs = state["agent_outputs"]
    
    # 檢測衝突
    conflict_detector = ConflictDetector()
    conflicts = conflict_detector.detect(agent_outputs)
    
    if not conflicts:
        return {"execution_trace": state["execution_trace"] + ["no_conflicts"]}
    
    # 解決衝突
    resolver = ConflictResolverAgent()
    resolution = await resolver.resolve_conflicts(
        recommendations=agent_outputs,
        user_context=state.get("user_profile", {}),
        conflicts=conflicts
    )
    
    return {
        "agent_outputs": resolution.resolved_outputs,
        "conflict_resolution": {
            "conflicts_found": len(conflicts),
            "resolution_method": resolution.method,
            "confidence": resolution.confidence
        },
        "execution_trace": state["execution_trace"] + ["conflicts_resolved"]
    }
```
